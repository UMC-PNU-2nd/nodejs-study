## 서브 도메인 이란?
서브 도메인은 보조 도메인으로써, URL로 전송하거나 계정 내의 IP 주소나 디렉토리로 포워딩되는 도메인 이름의 확장자이다

예를들어 네이버는 여러 서비스들을 아래와 같은 서브도메인을 통해 사용자가 접근할 수 있도록 한다.

네이버 블로그 : blog.naver.com
네이버 메일 : mail.naver.com
네이버 금융 : finance.naver.com
